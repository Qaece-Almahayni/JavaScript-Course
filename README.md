# JavaScript-Course


A compilation of core web development projects, architectural summaries, and reflections highlighting full-stack engineering competencies, programmatic problem-solving, and client-side scripting mastery.

## 🚀 Overview

This repository captures the technical progression through foundational web technologies. It bridges foundational computer science theory with production-ready software development by exploring semantic structuring, algorithmic layout design, and asynchronous behavioral programming.

## 🛠️ Tech Stack & Competencies

*   **HTML5:** Semantic document architecture, DOM structural hierarchy, and accessibility standards.
*   **CSS3:** Visual layouts, responsive design matrices, and interface styling.
*   **JavaScript (ES6+):** Event-driven logic, state tracking, DOM manipulation, and programmatic algorithmic loops.

## 📦 Core Artifacts

The following functional applications were built from the ground up to solidify key development paradigms:

### 1. Interactive Portfolio Website
*   **Description:** A centralized hub displaying technical capability, project deployment links, and clean UI design.
*   **Focus:** Responsive layout architectures, CSS layout systems, and clean navigation tracking.

### 2. Algorithmic Tic Tac Toe Game
*   **Description:** A classic logic game running entirely on client-side state engines.
*   **Focus:** Matrix state validation, win-condition evaluation algorithms, and event delegation.

### 3. Functional Calculator
*   **Description:** A mathematical engine parsing and executing arithmetic operations via UI input.
*   **Focus:** String parsing, edge-case handling (division by zero, decimal tracking), and state management.

### 4. Dynamic Pizza Menu
*   **Description:** An interactive e-commerce interface calculating runtime prices based on conditional item parameters.
*   **Focus:** Data collection arrays, cumulative summation algorithms, and real-time DOM updates.

## 🧠 Key Takeaways

*   **Engineered Problem Solving:** Developed a systematic approach to debugging, stack tracing, and resolving UI/logic mismatches.
*   **Framework Readiness:** Formed a deep comprehension of vanilla JavaScript execution contexts, serving as a prerequisite for frameworks like React, Vue, or Angular.
*   **Production Mindset:** Emphasized building complete, functional applications that address practical user experience and clean code workflows.
